# Traffic-monitoring-and-surveillance-system
This project presents an intelligent Traffic Monitoring and Surveillance System using deep  learning and computer vision techniques. The system is capable of detecting vehicles, monitoring  traffic flow, identifying over-speeding vehicles, and performing surveillance tasks such as  weapon detection and shoplifting detection. 
For traffic analysis, the system uses object tracking and speed estimation to detect vehicles 
crossing a defined region and identify speed violations. For surveillance, it detects people, bags, 
and suspicious activities such as weapon carrying and item theft from shelves. 
The system integrates multiple models including YOLO, RT-DETR, and a Hugging Face-based 
weapon detection model. A user-friendly interface is developed using Gradio to upload videos 
and visualize results. 
The results show that the system can accurately detect objects, track movement, generate alerts, 
and provide useful insights for security and traffic management.
